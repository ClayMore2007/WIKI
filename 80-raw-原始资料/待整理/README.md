# 待整理

这里是统一资料投放区。

你可以把临时资料先放到这里，然后让 LLM 按规则整理到对应目录。

## 可以放什么

- Markdown 笔记
- TXT 文本
- PDF
- 网页剪藏
- 图片或截图
- 股票研报
- 旅游攻略
- 技术文章
- 动画、电影、书影音资料
- 从 Cubox、浏览器、微信、邮件等导出的资料

## 建议命名

```text
YYYY-MM-DD-来源-主题.ext
```

示例：

```text
2026-05-10-cubox-日本关西旅行攻略.md
2026-05-10-pdf-英伟达财报.pdf
2026-05-10-web-ragflow项目介绍.md
```

## 整理后去向

- raw 来源笔记留在 `80-raw-原始资料/`
- 整理后的长期知识进入对应 wiki 目录
- 附件进入 `90-assets-附件/`
- 过期或不再使用的内容进入 `99-archive-归档/`
