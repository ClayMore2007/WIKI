# raw 原始资料

这里存放 raw 来源笔记。每篇文章、PDF、视频、文档或重要外部资料单独建一个文件。

raw 层保存“资料从哪里来、原文说了什么、有哪些可引用观点”；wiki 层保存“我整理后的长期知识和判断”。

## 统一投放区

新资料请先放到：

[[80-raw-原始资料/待整理/README]]

不要直接把大量资料粘贴到本 README。这个文件只作为 raw 层入口说明。

## 建议流程

1. 把新资料放进 `待整理/`。
2. 让 LLM 按 [[00-meta-规则/20-资料整理流程]] 整理。
3. raw 来源笔记留在 `80-raw-原始资料/`。
4. 长期知识沉淀到对应 wiki 目录。
