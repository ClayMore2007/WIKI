# 股票 raw 数据源

这里存放股票相关的长期更新数据源。

## 固定数据源

- `我的持仓.xls`：更新 [[10-股票投资/02-股票研究流/我的持仓]]
- `我的自选.xls`：更新 [[10-股票投资/02-股票研究流/我的自选]]

这两个 `.xls` 实际为 GBK 编码、制表符分隔文本。后续覆盖更新后，需要刷新对应 Markdown 页面。

## 展示字段规则

刷新 wiki 承接页时，默认剔除短期行情噪声字段：

- 涨幅
- 涨跌
- 自选时间
- 自选价格
- 自选收益
- 换手
- 主力净量
- 量比

这些字段仍保留在 raw 源文件中；需要短线复盘或行情核验时回到源文件或行情软件查看。

## 行业分类规则

刷新 wiki 承接页时，必须同步生成 `## 行业分类` 小节。

- `我的持仓`：按 `细分行业` 分组。
- `我的自选`：优先按 `所属行业` 分组；如果 `所属行业` 为空或为 `--`，回退到 `细分行业`；仍无可用行业时归入 `未标注`。
- 行业分类表使用 `行业 / 数量 / 标的` 三列。
- 行业分类用于快速观察组合暴露和自选池集中度，不替代正式行业研究。
