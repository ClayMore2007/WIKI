# 买入判断

这一块只放最终行动判断：买、等、卖、减仓、继续观察。

## 输入

- Cubox 资料流：[[10-股票投资/01-Cubox资料流/README]]
- 股票研究流：[[10-股票投资/02-股票研究流/README]]

## 输出

| 输出 | 入口 |
|---|---|
| 买入候选 | [[10-股票投资/03-买入判断/买入候选清单]] |
| 持仓处理 | [[10-股票投资/03-买入判断/持仓处理清单]] |
| 历史判断 | [[10-股票投资/03-买入判断/最终判断记录]] |

## 判断规则

- 只来自 Cubox 的观点不能直接买。
- 只有股票研究流里的单票分析，也要检查是否有外部资料或行业逻辑支持。
- 买入判断必须同时写清：买入理由、反证条件、仓位态度、复盘日期。
- 卖出或减仓判断必须写清：原假设哪里被破坏。

## 标准动作

| 动作 | 含义 |
|---|---|
| 买入 | 逻辑、估值、风险和触发条件都满足 |
| 观察 | 逻辑成立但价格、财报或关键事实未满足 |
| 不买 | 逻辑弱、估值过高、风险不可控或资料未核验 |
| 减仓 | 原逻辑仍在但风险暴露过高 |
| 卖出 | 原假设被破坏或出现更优替代 |
