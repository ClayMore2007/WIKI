# 行业

这里存放行业主题、产业链框架和跨公司比较。行业页优先沉淀可复用框架，不直接承接未经核验的短视频结论。

## 资料状态流

行业线索按下面状态推进，未进入 `verified` 前不直接用于买卖判断。

| 状态 | 含义 | 可进入决策 |
|---|---|---|
| inbox | 刚进入的标题、链接、视频、碎片笔记 | 否 |
| extracted | 已转写或提取出公司、产业链、关键主张 | 否 |
| verified | 已用公告、财报、研报或可信数据核验关键事实 | 可以作为依据之一 |
| rejected | 核验失败、与股票投资弱相关或噪声过高 | 否，归档或迁移 |
| archived | 暂无研究价值但保留历史记录 | 否 |
