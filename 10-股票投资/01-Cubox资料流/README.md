# Cubox 资料流

这一块只处理来自 `80-raw-原始资料/Cubox/股票/` 的外部资料，包括短视频、文章、社媒观点和产业链线索。

## 目标

把外部资料整理成可参考的信息，而不是直接变成买卖建议。

## 流程

1. 从 raw/Cubox 进入。
2. 进入 [[10-股票投资/01-Cubox资料流/Cubox资料索引]]。
3. 提取主题、公司、数据、风险和待核验问题。
4. 进入 [[10-股票投资/01-Cubox资料流/主题资料总结/README]] 或 [[10-股票投资/01-Cubox资料流/公司线索/README]]。
5. 只有核验后的信息才能进入 [[10-股票投资/03-买入判断/买入候选清单]]。

## 状态

| 状态 | 含义 |
|---|---|
| inbox | 刚进入，只有标题、链接或标签 |
| extracted | 已提取核心信息 |
| verified | 已用公告、财报、研报或可信数据核验 |
| rejected | 标签错误、事实不成立或与投资弱相关 |
| archived | 保留历史记录，不再跟踪 |
