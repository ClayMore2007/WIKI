# 公司

每家公司一页。

## 职责边界

- `02-股票研究流/公司档案/`：长期公司档案，记录业务模式、历史财务、产业链位置、关键风险，不随每次行情快照频繁改写。
- `02-股票研究流/持仓分析/`：某个日期的持仓判断快照，记录当时的评级、仓位态度、卖出条件和下次复盘日期。
- [[10-股票投资/03-买入判断/持仓处理清单]]：行动总表，只放当前需要跟踪或处理的结论。

## 分析规范

- 单只股票深度分析优先使用 [[10-股票投资/02-股票研究流/分析框架/股票深度分析模板-7步法]]。
- 写法参考 [[10-股票投资/02-股票研究流/分析框架/案例-中天科技-600522-7步法]]。
- 股票页必须写清数据日期、来源、不确定性和触发条件。
- 完成公司页或持仓分析页后，把核心假设、状态和下次复盘日期同步到 [[10-股票投资/03-买入判断/持仓处理清单]]。
