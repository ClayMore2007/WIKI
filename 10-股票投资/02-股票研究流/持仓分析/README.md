# 持仓分析

这里存放单只持仓的日期快照和阶段性分析。组合层汇总看 [[10-股票投资/02-股票研究流/我的持仓分析]]，最终行动看 [[10-股票投资/03-买入判断/持仓处理清单]]。

- 新增或更新持仓分析后，同步检查 [[10-股票投资/02-股票研究流/我的持仓分析]]。
