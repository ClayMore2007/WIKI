# 股票研究流

这一块处理你的真实股票池：持仓、自选、单票分析、公司档案和分析框架。

## 目标

把“我现在持有什么”和“我可能想买什么”分析清楚，形成可被复盘的单票判断。

## 流程

1. 从 [[10-股票投资/02-股票研究流/我的持仓]] 和 [[10-股票投资/02-股票研究流/我的自选]] 进入。
2. 对重点股票建立单票分析，放入 [[10-股票投资/02-股票研究流/持仓分析/README]] 或 [[10-股票投资/02-股票研究流/自选分析/README]]。
3. 长期不随行情频繁变化的内容沉淀到 [[10-股票投资/02-股票研究流/公司档案/README]]。
4. 组合层判断放在 [[10-股票投资/02-股票研究流/我的持仓分析]]。
5. 最终动作进入 [[10-股票投资/03-买入判断/README]]。

## 单票分析必须回答

| 问题 | 说明 |
|---|---|
| 为什么关注 | 主题、行业、公司或估值逻辑 |
| 现在贵不贵 | 当前估值和前瞻估值 |
| 最大风险是什么 | 能推翻判断的关键变量 |
| 买入条件 | 价格、估值、财报或催化条件 |
| 卖出条件 | 业绩、估值、趋势或风险触发 |
| 下次复盘日期 | 明确下一次检查时间 |
