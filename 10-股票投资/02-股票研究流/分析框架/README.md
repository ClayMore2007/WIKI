# 股票分析框架

这里存放股票分析的通用模板、示例案例和后续可复用的方法论。用于把单只股票研究统一成“事实 / 假设 / 风险 / 触发条件 / 复盘”的结构，避免只看短期行情或单一视频结论。

## 核心入口

- [[10-股票投资/02-股票研究流/分析框架/股票深度分析模板-7步法]]
- [[10-股票投资/02-股票研究流/分析框架/案例-中天科技-600522-7步法]]

## 使用方式

1. 新分析一只股票时，复制 [[10-股票投资/02-股票研究流/分析框架/股票深度分析模板-7步法]] 到 `10-股票投资/02-股票研究流/公司档案/`。
2. 文件名建议使用 `公司名-股票代码-深度分析`。
3. 填写时参考 [[10-股票投资/02-股票研究流/分析框架/案例-中天科技-600522-7步法]] 的结构，但每个数据点都要补来源。
4. 完成后，把核心结论和下次复盘时间同步到 [[10-股票投资/03-买入判断/持仓处理清单]]。

## 维护规则

- 股票分析默认是高风险内容，必须保留来源、日期和不确定性。
- 案例中的财务、估值和价格数据会过期，正式决策前必须回到公告、财报、行情软件或研报复核。
- “总判”和“最终结论”必须能被“风险排查”和“触发条件”反驳，不写单边乐观结论。
