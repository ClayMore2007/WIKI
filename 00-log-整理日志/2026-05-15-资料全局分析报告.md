## 二、全局判断

当前资料库最有价值的两条主线是：

1. **AI 算力链 / 半导体 / 高端材料投研线**  
   资料密度最高，已有多条完整转写和主题页。它能形成一条从 AI 算力需求、芯片、光模块、PCB、液冷、电源、超节点、国产替代、高端材料到财报验证的研究链。但这条线风险也最高，几乎全部来自短视频、社媒和二手研报，需要回公告、财报、原研报和可信数据库核验。

2. **AI 工具 / Claude Code / Obsidian 工作流线**  
   资料数量多，和本知识库维护方式高度相关。现有资料已经沉淀出 Harness Engineering、Obsidian-Claude-Code 内容创作自动化、TradingAgents 等页面，但大量 Claude Code / Skill / Agent 视频仍只是 raw 标题或 metadata。下一步适合转成“可执行工作流清单”和“工具实践规范”。

其余资料更像候选池：

- 旅游资料适合整理成“成都周边/川西候选目的地”，但必须核验地点、开放状态、费用、交通和季节。
- 健康资料只能做低置信索引，不能直接沉淀功效或疗法。
- 保险资料已经处理得足够克制，适合保持低敏索引。
- 未分类资料不宜继续扩散到主知识页，除非用户确认用途。

### 股票资料的核心风险

- **来源风险**：短视频和社媒内容不能替代原研报、公告、财报。
- **ASR 风险**：公司名、金额、百分比、技术缩写经常误识别。
- **题材风险**：AI 算力链、高端材料、国产替代容易把“产业趋势”误读成“个股机会”。
- **重复叙事风险**：光模块、PCB、液冷、IDC、超节点等线索多次出现，容易强化单一主题偏见。
