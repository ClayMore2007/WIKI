# 同花顺股票更新规则

本页是同花顺股票更新的唯一规则入口。解析、字段映射、代码归一化、Markdown 生成和校验细节由脚本负责，不在规则里重复维护。

## 触发条件

| 用户意图 | 执行命令 |
|---|---|
| 更新股票 / 更新我的股票 / 同步同花顺 | `python .\Tools-工具\update_ths_stocks.py` |
| 只更新自选股 / 同花顺自选股 | `python .\Tools-工具\update_ths_stocks.py --only self` |
| 只更新持仓 / 我的持仓 / 同花顺持仓板块 | `python .\Tools-工具\update_ths_stocks.py --only holdings` |
| 表格整理股票信息 / 输出股票 Markdown 汇总 | 先确保 `我的自选.md` 和 `我的持仓.md` 已更新，再更新 `10-股票投资/02-股票研究流/我的股票信息.md` |

## 安全边界

- 只读本地同花顺缓存、名称表和“我的板块”文件，不修改同花顺目录。
- 不读取交易记录、资金流水、账号、密码、券商登录、银行卡等高敏信息。
- 不操作 `ClayMore-Private-Wiki/`。
- 股票和财务内容视为高风险，必须标注来源和不确定性。
- 自选股清单只表示观察列表；“我的持仓”来自同花顺自定义板块，不等同于券商真实持仓。

## 输入输出

| 类型 | 路径 |
|---|---|
| 脚本 | `Tools-工具/update_ths_stocks.py` |
| 脚本测试 | `Tools-工具/test_update_ths_stocks.py` |
| 自选股目标页 | `10-股票投资/02-股票研究流/我的自选.md` |
| 持仓目标页 | `10-股票投资/02-股票研究流/我的持仓.md` |
| 股票信息汇总页 | `10-股票投资/02-股票研究流/我的股票信息.md` |
| raw 快照 | `80-raw-原始资料/同花顺自选股/`、`80-raw-原始资料/同花顺持仓板块/` |
| 整理日志 | `00-log-整理日志/` |

## 执行要求

1. 在 Obsidia 根目录运行脚本。
2. 交付前运行 `python .\Tools-工具\test_update_ths_stocks.py`。
3. 向用户报告自选股和持仓的记录数、名称缺失、价格/日期缺失、新增、移除。
4. 用户要求 Markdown 表格汇总时，优先更新已有的 `我的股票信息.md`；若不存在，可创建该页，至少包含更新摘要、我的持仓、自选股、待补充信息和相关页面。
5. 汇总页只整合脚本目标页已有字段，不补写行业、估值、仓位、成本、盈亏等缺源或高敏字段。
6. 如果脚本报错，修脚本或数据源问题；不要把脚本已经覆盖的解析细节补回规则页。

## 不确定性声明

每次输出股票清单时附上或确认目标页中已有：

> 来源为本机同花顺缓存和自定义板块；价格和日期是本地记录值，可能不是实时行情。该清单仅用于个人观察和 Wiki 记录，不构成投资建议。
