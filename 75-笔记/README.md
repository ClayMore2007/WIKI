# 笔记

这里放日常随手笔记。

## 目录

- `00-收件箱/`：用户以“笔记”触发保存的日常笔记。
- `10-已整理/`：已经整理过、并在正文中标注去向的笔记。

## 使用方式

用户消息以“笔记”开头时，默认将正文保存为一篇笔记，放入 `00-收件箱/`。

用户说“整理笔记”时，从 `00-收件箱/` 读取笔记，按质量和主题整理到 wiki：优先更新已有页面，必要时创建主题页、问题页或决策页；整理过程写入 `00-log-整理日志/`。
